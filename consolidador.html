Consolidador de datos para Pre Tests y Post Tests
Modo de uso:
  1. Presionar Examinar
  2. En la ventana emergente, elegir los archivos a cargar. Cada archivo deberá corresponder a un cuestionario.
    Notar que manteniendo presionado Shift se pueden elegir grupos de archivos.
    También se puede mantener presionada la tecla Ctrl para elegir individualmente varios archivos a la vez.
  3. Click en "Abrir" o "Aceptar"
  4. La lista se cargará, y podrá ser copiada manualmente al Google Doc correspondiente
  5. Si faltó algún archivo, puede presionarse nuevamente "Examinar..."
